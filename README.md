# wagergames

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint

**IF ENCOUNTERING ERRORS**

First,

npm un node-sass 
// or
yarn remove node-sass
Then,

npm i node-sass --save-dev
// or
yarn add node-sass --dev


THEN 
npm install


```
